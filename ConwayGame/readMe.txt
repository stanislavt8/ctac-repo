================================
 Gaming framework
================================

This is an attempt to create a Gaming framework to allow to easy implement a variety of games
working with 2-D, 3-D, or n-D matrix.

The first and only implementation for now is Conway's game of life on 2-D matrix
 

=============
Prerequisites
=============

* Java 1.7 or later
* Maven 3.x+

=================================
Building and Running instructions
=================================

This is a pretty standard maven project, so use standard maven commands like:

* mvn clean install 	- to biuld jar

To run a simple demo program:

...target>java -jar conway-game-0.0.1-SNAPSHOT-jar-with-dependencies.jar
